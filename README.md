A secure backend application built using Spring Boot for managing investment buy/sell transactions.
The system supports JWT-based authentication, role-based authorization, transaction management, and streaming Excel export for large datasets.
This project demonstrates real-world backend concepts such as security, pagination, exception handling, testing, and performance-optimized data export.

Tech Stack:-
Java 17+
Spring Boot
Spring Security
JWT (JSON Web Tokens)
Spring Data JPA (Hibernate)
PostgreSQL
Apache POI (SXSSF – Streaming Excel)
JUnit 5 & Mockito
Maven
